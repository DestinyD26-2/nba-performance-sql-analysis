NBA Performance SQL Analysis

📊 Overview
This project analyzes NBA game data (2004–2020) to identify trends in scoring, home-court advantage, and the impact of three-point shooting on team success.

🎯 Objective
To understand how gameplay and strategy have evolved over time and what factors contribute to winning.

🛠️ Tools Used
- SQL
- Dataset: NBA Games (2004–2020)

🔍 Key Questions
- How have scoring trends changed over time?
- Is home-court advantage declining?
- How important is three-point shooting for winning?

📈 Key Insights
- Average points scored have increased over time
- Home team win rate has slightly declined
- Teams with higher 3-point percentages tend to win more games
- Very few teams with high 3-point accuracy have losing records

📂 Project Structure
- `queries.sql` → SQL queries
- `insights.md` → analysis and conclusions

🚀 Skills Demonstrated
- Aggregations (AVG, COUNT, SUM)
- GROUP BY and HAVING
- Subqueries
- Data trend analysis

💡 Key Takeaway
Three-point shooting plays a significant role in team success, but it is not the only factor—other metrics like free throws and possession should also be considered.
